# ЗнатокOS

Операционная система на Lua для мода **CC: Tweaked** (Minecraft). Работает на Advanced Computer (цветной экран, мышь).

## Возможности

- Многозадачный шелл с историей, alias и автодополнением
- Оконный менеджер с taskbar (Ctrl+Tab для переключения)
- Графический рабочий стол с иконками и контекстным меню
- Редактор с подсветкой Lua
- Пользователи и пароли (SHA-256)
- Сеть: чат и передача файлов через rednet
- Пакетный менеджер (HTTP-репо + встроенный каталог)
- Темы оформления
- Демо-приложения: часы, калькулятор, змейка, paint, файловый менеджер

## Установка

В игре на Advanced Computer:

```
wget run https://raw.githubusercontent.com/<your-repo>/os/installer.lua
```

Или вручную скопировать `installer.lua` на компьютер и выполнить:

```
installer
```

После установки — `reboot`.

## Горячие клавиши

- **Ctrl+Tab** — переключение окон
- **Ctrl+W** — закрыть окно
- **Ctrl+N** — новый терминал (на рабочем столе)
- **F10** — открыть меню «Пуск»

```
│   └── apps/         — приложения
└── tests/            — ручные тесты
```

## Кириллический шрифт

CC: Tweaked по умолчанию не поддерживает кириллицу. В папке
[`resourcepack/`](resourcepack/) — генератор и инструкция по сборке
ресурспака Minecraft с кириллическим шрифтом. ОС автоматически
перекодирует UTF-8 → CP1251 на выводе (см. `src/util/cyrillic.lua`),
так что пара «ресурспак + ОС» даёт читабельный русский.

Если ресурспак не ставить, вывод будет оставаться нечитабельным —
функционал ОС при этом не страдает (команды принимают латиницу).
